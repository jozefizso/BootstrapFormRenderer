BootstrapRenderer is an extension for Nette web applications for rendering
forms using Bootstrap styling.

BootstrapRenderer targets Nette framework 2.3 and the Bootstrap CSS v2.x.
Library must work on PHP 5.6, 7.0 and 7.1 runtimes.

Use Context7 to access documenatation about Nette 2.3, Latte 2.3 and Bootstrap v2.
Ensure the documentation is for the versions used in this project.
