BootstrapRenderer is an extension for Nette web applications for rendering
forms using Bootstrap styling.

BootstrapRenderer targets Nette framework 2.1 and the Bootstrap CSS v2.x.
Library must work on PHP 5.6 and PHP 7.0 runtimes.

Use Context7 to access documenatation about Nette and Bootstrap.
Ensure the documentation is for the versions used in this project.
